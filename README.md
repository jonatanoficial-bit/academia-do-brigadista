# Site do Curso de Brigadista (GitHub Pages)

Este é um site estático (HTML/CSS) pronto para subir no GitHub Pages.

## Como publicar
1. Crie um repositório no GitHub (ex.: `BRIGADISTA-1.0`).
2. Envie (upload) todos os arquivos desta pasta para a raiz do repositório.
3. No GitHub: **Settings → Pages**.
4. Em **Build and deployment** selecione:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (ou `master`) e **/(root)**
5. Salve. O GitHub vai mostrar o link do seu site.

## Personalizar
- Troque o banner em `assets/img/banner.png`.
- Edite os textos dentro de `index.html` e `paginas/*.html`.
- Se quiser incluir mapas/figuras, coloque imagens em `assets/img/` e referencie nas páginas.
